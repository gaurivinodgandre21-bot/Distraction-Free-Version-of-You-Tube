# 🎥 Distraction-Free YouTube

A simple YouTube-inspired webpage built using **HTML** that provides a clean, distraction-free viewing experience by displaying only the essential content. This project was created to practice HTML concepts such as page structure, hyperlinks, images, and semantic elements.

## 🚀 Features

- Clean and minimal YouTube-inspired layout
- Focused interface with reduced distractions
- Navigation between different categories using HTML pages
- Beginner-friendly project
- Built using only HTML

## 🛠️ Technologies Used

- HTML5

## 📂 Project Structure

```
Distraction-Free-YT/
│── index.html
│── art.html
│── gaming.html
│── music.html
│── education.html
│── ...other HTML files
```

## 🎯 Purpose

The goal of this project was to strengthen my understanding of HTML by creating a multi-page website inspired by YouTube while keeping the interface simple and distraction-free.
